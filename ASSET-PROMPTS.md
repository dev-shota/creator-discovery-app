# 画像素材プロンプト集（GPT image 2 用）

> 動画「Claude Design × GPT image 2」の手法に沿った、この app 用の画像生成プロンプト。
> **現状アプリは CSS グラデーションの有機シェイプを標準装備**しているので、これらの画像が無くても成立する。
> 「素材がデザインの良さを決める」（動画の主旨）ので、よりリッチにしたい時にここで素材を作って差し替える。
>
> パレットは PLAY/GROUND 系（実装と一致させること）:
> pink `#ED3A8C` / yellow `#EFF33C` / teal `#2DC7C0` / purple `#A78BFA` / 背景 白 / インク `#2A2A33`

## 使い方（動画のワークフロー）

1. 下の **(A) メインビジュアル** か **(B) 素材シート** を GPT image 2 に投げて、**大きめ（高解像度）**で生成する。
2. 素材シートは **(C) の指示で Claude（または Claude Code）に「1枚ずつ分割＋透過」**させる（GPT は分割が苦手）。私に渡してくれれば私が分割・透過まで実行する。
3. 透過 PNG を `public/assets/` に置く。ファイル名を教えてくれれば、私が `pages/index.vue` / `assets/main.css` の装飾に配線する（今の CSS ブロブと差し替え or 併用）。

---

## (A) メインビジュアル / デザイン画像（方向性決め）

```
あなたはプロの Web デザイナーです。次の仕様で、Web サイトのヒーロー（ファーストビュー）デザインを1枚の画像として作成してください。

# 目的・業種
「好きな漫画家・原作者から、その作品とアニメ化スタジオをたどる」discovery ツールの紹介ページ。
個人開発の Web アプリ（非商用・非公式）。

# ターゲット
漫画・アニメが好きな10〜30代。新しい作品や“次に読むもの”を探している人。

# トンマナ（重要・出力に強く影響）
遊び心がある / 親しみやすい / ポップ / クリーン / AI 感がない（テンプレ臭くない）。
柔らかく有機的な抽象シェイプを多用。余白は広め。角丸。フラット。

# デザイン要件
- 配色: 白背景に、pink #ED3A8C・yellow #EFF33C・teal #2DC7C0・purple #A78BFA のパステル寄りアクセント。
- 不規則で柔らかい抽象シェイプ（ブロブ・波・円）を背景に散らす。ふわっと浮いている雰囲気。
- 丸みのあるサンセリフ見出し。手書き風スクリプトのワンポイント。
- 大きな見出しテキストのスペースを左に確保（テキストは後で差し替えるのでダミーで可）。
- 右側はメインビジュアル（イラストやシェイプ）のスペースを確保＝差し替え可能な余白にする。

横長（16:9 以上、1280×720px 以上）で1枚生成してください。
```

## (B) 素材シート（背景・装飾アセットをまとめて抽出）

```
次の Web デザイン用の素材を、1枚の画像にまとめて出力してください。

# 内容
- 柔らかく有機的な抽象シェイプ（ブロブ）8〜12個。pink #ED3A8C / yellow #EFF33C / teal #2DC7C0 / purple #A78BFA のパステル〜鮮やかの濃淡。
- 波形（セクション区切り用の wave）2〜3種。
- 小さな装飾（星・きらめき・点線・円）数種。

# レイアウト（切り出ししやすさが最重要）
- **正方形のグリッド**（例 4×3）に、1マス1素材で**独立**して配置する。
- 素材同士が重ならない・くっつかないようにする。
- グリッドは**ピクセル単位を揃えた正方形セル**にする（例 1024×1024 を 4×3 等分）。
- 背景は白（あとで透過しやすいよう、素材の縁をはっきりさせる）。

できるだけ**高解像度**で1枚生成してください。
```

---

## (C) 切り出し・透過（Claude / Claude Code に渡す指示）

```
この素材シート画像を、グリッドの1マスごとに1枚ずつ分割し、それぞれ背景を透過した PNG にしてください。
- 各セルの境界（正方形グリッド）でクロップする。
- 白背景を透過にする（アンチエイリアスの縁は残す）。
- 連番のファイル名（asset-01.png …）で public/assets/ に保存する想定で出力してください。
```

## (D) ヒーロー用イラスト（1点もの・AI感を抑える版）

> AI感の主因＝①エアブラシ/グラデのにじみ塗り ②「思考の雲に浮かぶキャラ」等の既視感構図
> ③ブランド外のgeneric パステル ④詰め込み構図。下のプロンプトはそれを潰す（フラット塗り・
> 厳密パレット・余白・透過）。

```
フラットなベクターイラスト1枚。テーマ「漫画の作り手と、その作品」。

# スタイル（AI感を消す肝）
- フラットデザイン: 単色のベタ塗り、グラデーション/エアブラシのにじみ無し、均一な細い輪郭線。
- アニメ風エアブラシ塗りや写実は避ける。手描きのわずかな揺らぎのある線（完璧すぎない＝意図的な温かみ）。
- 厳密に限定したパレットのみ: ピンク #ED3A8C / ティール #2DC7C0 / イエロー #EFF33C /
  パープル #A78BFA / インク #2A2A33 / 白。これ以外の色は使わない。
- 余白を広く取り、被写体は中央でなく片側へ寄せる（詰め込まない）。要素は少なめ。

# 構図（既視感のある表現を避ける）
- 漫画家が机でペンを走らせ、その線が1本のリボンのように流れて、開いた本（＝作品）や
  四角い画面（＝アニメ）へつながる、というシンプルな比喩。
- 「思考の雲に浮かぶキャラクター」のような既視感構図は使わない。背景は描かない。

# 出力
- 背景は透過（PNG）。被写体だけを切り抜いた状態。文字は入れない。正方形〜縦長・1024px 以上。
```

> 注意（法務）: ここで作る素材は**抽象シェイプ／オリジナルイラスト**に限る。漫画のカバー画像・キャラクター・ロゴは生成・再ホストしない（カバーは AniList hotlink のみ＝README の不変条件）。
